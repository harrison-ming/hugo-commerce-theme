组件参数说明

参数 类型 默认值 说明
layoutstring"list"list | grid | cards
columnsint2列数（grid/cards）
showAvatarbooltrue显示头像
showRatingbooltrue显示星级
featuredOnlyboolfalse只显示精选
limitint999限制数量

核心特性

  • ✅
    数据驱动

    所有评价数据集中在 data/testimonials.toml

  • ✅
    3种布局

    List（列表）、Grid（网格）、Cards（大卡片）

  • ✅
    响应式设计

    自动适配移动端、平板、桌面

  • ✅
    灵活过滤

    支持精选筛选、数量限制
