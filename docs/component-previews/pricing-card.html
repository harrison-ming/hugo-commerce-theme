所有定价信息都存储在 TOML 文件中，便于管理和更新

[[plans]] id = "single" name = "Single Decree" icon = "🥉" tagline = "Perfect for personal blessing" originalPrice = 198 price = 99 currency = "$" save = 99 features = [ "1 Celestial Decree", "Handcrafted & Blessed", "Divine Protection", "Free Worldwide Shipping" ] ctaText = "👉 Claim Your Decree Now" ctaLink = "/checkout" ctaTheme = "green" [[plans]] id = "family" name = "Family Protection Pack" icon = "🥈" tagline = "Bless your home and loved ones" originalPrice = 396 price = 169 save = 227 saveNote = "Best Value!" badge = "MOST POPULAR!" badgeColor = "blue" ctaTheme = "blue" highlight = true # ... features ... [[plans]] id = "ultimate" name = "Ultimate Abundance Pack" icon = "🥇" originalPrice = 594 price = 249 save = 345 saveNote = "Ultimate Blessing!" badge = "MAXIMUM ENERGY!" badgeColor = "purple" ctaTheme = "purple" # ... features ...

组件参数说明

参数 类型 默认值 说明
plans slice site.Data.pricing.plans 定价方案数组
layout string "card" card | table | minimal
columns int 3 2 | 3 | 4（列数）

定价方案字段说明

字段 类型 必需 说明
idstring是唯一标识符
namestring是产品名称
iconstring否emoji 图标
taglinestring否副标题
originalPriceint否原价（划线）
priceint是当前价格
currencystring是货币符号
saveint否节省金额
saveNotestring否节省说明
badgestring否徽章文字
badgeColorstring否blue | purple | green
highlightbool否是否高亮边框
featuresarray是功能列表
ctaTextstring是按钮文字
ctaLinkstring是按钮链接
ctaThemestring否green | blue | purple

核心特性

  • ✅
    数据驱动

    所有定价信息集中在 data/pricing.toml，便于管理和更新

  • ✅
    响应式设计

    自动适配移动端、平板、桌面，1/2/3 列布局

  • ✅
    多主题支持

    绿色/蓝色/紫色按钮主题，匹配不同产品定位

  • ✅
    徽章系统

    支持"MOST POPULAR"、"MAXIMUM ENERGY"等徽章，突出重点方案

  • ✅
    价格对比

    原价划线 + 当前价 + 节省金额，清晰展示优惠

  • ✅
    悬停效果

    scale-105 + 阴影增强，提升交互体验
