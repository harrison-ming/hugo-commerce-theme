Product Gallery Component

产品图片画廊组件 - 多种展示布局

← 返回组件列表

Thumbnail Layout（缩略图布局）

经典的产品页布局：大主图 + 缩略图切换

Shortcode 用法：

{{< product-gallery >}} # 显示图片标题 {{< product-gallery showCaptions=true >}}

Grid Layout（网格布局）

网格展示所有图片，适合浏览产品细节

Shortcode 用法：

{{< product-gallery layout="grid" >}} # 4列网格 {{< product-gallery layout="grid" columns=4 >}} # 2列网格 {{< product-gallery layout="grid" columns=2 >}}

Carousel Layout（轮播布局）

横向滚动轮播，移动端友好

Swipe to see more

Shortcode 用法：

{{< product-gallery layout="carousel" >}} # 带标题的轮播 {{< product-gallery layout="carousel" showCaptions=true >}}

组件参数说明

参数 类型 默认值 说明
layoutstring"thumbnail"thumbnail | grid | carousel
columnsint3网格列数（2|3|4）
showCaptionsboolfalse显示图片标题
aspectRatiostring"square"square | landscape | portrait

核心特性

  • ✅
    3种布局

    Thumbnail（缩略图）、Grid（网格）、Carousel（轮播）

  • ✅
    交互切换

    缩略图布局支持点击切换主图

  • ✅
    响应式设计

    自动适配移动端、平板、桌面

  • ✅
    CSS Scroll Snap

    轮播使用原生滚动，无需 JavaScript

  • ✅
    数据驱动

    所有图片集中在 data/product-gallery.toml

  • ✅
    悬停效果

    网格布局图片放大、边框高亮
